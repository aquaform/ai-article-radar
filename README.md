# 📡 AI Article Radar

Веб-приложение, которое мониторит три русскоязычные площадки — **VC.ru**, **Habr** и **Дзен** — находит свежие материалы про ИИ/нейросети, рекомендует их и показывает короткое эвристическое саммари (без LLM/платных API).

**Публичный URL:** https://ai-article-radar.apps.sielom.ru

## Возможности
- Лента свежих AI-статей с источником, заголовком, ссылкой, датой, саммари, тегами и оценкой релевантности.
- Статусы статей (`Новое`, `Хочу прочитать`, `Читаю`, `Прочитано`, `Не интересно`, `Архив`) с сохранением в `localStorage`.
- Фильтры по источнику и статусу, поиск, кнопка «Обновить», быстрое скрытие «Не интересно».
- Экспорт/импорт статусов JSON и экспорт списка «Хочу прочитать» в Markdown.
- Mobile-friendly интерфейс.
- Серверный агрегатор с кэшем 15 минут и честной диагностикой источников (degraded-режим вместо фейков).

## Источники
| Площадка | Метод |
|----------|-------|
| Habr | RSS хаба «Искусственный интеллект» |
| VC.ru | Общий RSS + фильтр по AI-ключевым словам |
| Дзен | Публичный JSON-эндпоинт ленты + фильтр по AI-ключевым словам |

Если источник недоступен или блокирует — приложение продолжает работать с остальными и явно показывает degraded-статус. Фейковые статьи не добавляются.

## Запуск локально
```bash
npm install
npm run build
npm start            # http://localhost:3000
# или в dev-режиме:
npm run dev
```

## Тесты
```bash
npm test
```

## API
- `GET /health` → `200 ok`
- `GET /api/articles` → `{ articles, diagnostics, lastUpdated, cacheAgeSeconds, degraded }`
- `GET /api/articles?refresh=1` → принудительное обновление кэша

## Стек
Node.js 20 · TypeScript · Express · fast-xml-parser · vitest · vanilla SPA. Docker-образ собирает TypeScript и запускает `node dist/server.js` на `0.0.0.0:$PORT`.
